Local Storage vs Session Storage
Local Storage
  • Data is stored permanently in local storage.
  • It must be deleted via javascript or manually.
  • Local storage has 4 methods: 1) setItem() 2) getItem() 3) removeItem() 4) clear()
  • Syntax : myStorage = window.localStorage;
Session Storage
  • Data is stored temporary in session storage.
  • It's session-based and works per window or tab.
  • Session Storage has 4 methods: 1) setItem() 2) getIteam() 3) removeItem() 4) clear()
  • Syntax : myStorage = window.sessionStorage;
